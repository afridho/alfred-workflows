# Arti Kata (KBBI) - OFFLINE :book:

Cari arti kata di KBBI menggunakan keyword. 
Workflow ini menggunakan database sendiri (didalam folder workflow), jadi tidak membutuhkan koneksi internet.

| Command            | Deskripsi                   |
